Description :

Ce mini-projet a été réalisé dans le cadre d’un travail pratique de cours.
L’objectif est de créer une activité interactive d’apprentissage autour du tri des déchets, en utilisant Unity 3D.

Le joueur peut attraper et déplacer différents déchets (bouteilles, papiers, canettes, aliments) et les déposer dans les poubelles correspondantes.
Une validation visuelle (V ou F) indique si le tri est correct ou non.


Poubelles colorées selon le type de déchet :

Papier → Bleu

Bouteilles → Vert

Cannettes → Jaune

Alimentation → Marron

Interaction Drag & Drop : déplacer un objet et le déposer dans une poubelle.

Feedback visuel : V pour une bonne réponse, F pour une erreur.
